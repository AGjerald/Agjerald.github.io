Prosjektet

Studentorganisasjonen «Springbrettet» har som mål å redusere avstanden mellom studenter og potensielle arbeidsgivere. Jeg ble rekruttert som webutvikler og fikk stor frihet til å designe og utvikle nettsiden i tråd med organisasjonens mål og identitet.

Nettsiden er bygget og driftet på dra-og-slipp-verktøyet SquareSpace. Selv om utviklingen går raskt med et slikt verktøy, la jeg inn egen HTML, CSS og JavaScript for å implementere interaktive, dynamiske og egenutviklede komponenter.
